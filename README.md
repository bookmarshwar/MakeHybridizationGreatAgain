### 机器 (Machine)
- **AutomatedBreedingFacility（自动化育种设施）**：核心自动化杂交设备，提升作物育种效率

### Mixin 改造（IC2 兼容）
- 作物新增属性 `overclock`（超频）：作为第四维度属性，唯一作用是提升最终产量
- 产量计算公式：`最终产量 k = n ^ ((ov - 1) * 1.1)`  
  （其中 `n` 为 IC2 原产量，`ov` 为作物 `overclock` 属性值）
- 兼容 IC2 原有种植逻辑，无额外配置即可生效
